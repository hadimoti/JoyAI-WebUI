# 🦊 Joy AI

Responsive phone/desktop WebUI for the Hermes Agent — restaurant ops, marketing, IG posts & content automation.

## Features
- 4 authorized users with role-based access
- Gmail verification (admin-approved emails) **or** Telegram verification
- Model filter (default model preconfigured)
- localStorage caching: user / models / histories
- Dark-grey + orange theme, animated fox 🦊
- Fully responsive (phone → tablet → desktop)
