# Joy AI 🦊

A responsive phone/desktop WebUI for the **Hermes Agent** system.
Dark-grey + orange theme with an animated fox mascot.

## Features
- 4 authorized users (name-based login)
- Gmail verification (admin-approved emails) OR Telegram verification
- Per-user chat history (cached locally)
- Model filter (default model system)
- Fully responsive: phone → tablet → desktop
- Demo mode (works with zero backend)

## Quick Start

### Frontend (test now)
Open `frontend/index.html` in any browser.
`DEMO_MODE = true` lets you test the full flow:
- Gmail code = `123456`
- Telegram auto-confirms

### Backend (when ready)
See `docs/SETUP.md`.
